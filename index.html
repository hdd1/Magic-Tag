Magic Tag

Простой в использовании, с широким набором возможностей

MagicTag очень гибко настраивается. На свой вкус можно менять лейбл, плейсхолдер, размер выпадающего окна, местоположение и наличие поиска, делать выбор с разделителем, делать предварительный выбор. 

$('#selectCity').magicTags({
    placeholder: 'Выберите город',
    label: 'Город',
    data: [
        {id: 1, name: 'Лондон'},
        {id: 2, name: 'Париж'},
        {id: 3, name: 'Берлин'},
        ...
    ],
});

С разделителями

data: [
    {divider: 'Европа'},
    {id: 1, name: 'Лондон'},
    {id: 2, name: 'Париж'},
    ...
]

С явной строкой поиска

searchInListBox: true,
searchPlaceholder: 'Начните набирать название',

Предварительный выбор

selected: [1,3],

Выбор только одного значения

multiselect: false,

Параметры

Параметр Тип Умолч Описание
placeholder string Текст, который отображается пока не выбран ни один тег
label string Заголовок блока тегов
search bool true Будет ли поиск
searchInListBox bool false Если true, то поиск будет в выпадающем списке, иначе в блоке с тегами
searchPlaceholder string Плейсхолдер в строке поиска (работает, если searchInListBox установлен в true)
ifOneSelected bool true Если список элементов содержит всего одно значение - автоматически будет выбрано по-умолчанию
multiselect bool true Режим переключателя: можно ли выбирать несколько элементов списка или только один
listBoxMaxHeight int h/2 Максимальный размер выпадающего списка. По-умолчанию - половина размера окна браузера
selected array id выбранных элементов списка
data object Список элементов 
data: [
    {id: 1, name: 'Лондон'},
    {id: 2, name: 'Париж'},
    ...
]
